### Ontology

During our study we considered the 4 datasets presented in [Ontologue: Declarative Benchmark Construction for Hierarchical Multi-Label Classification](https://proceedings.neurips.cc/paper_files/paper/2022/file/8cf04c64d1734e5f7e63418a2a4d49de-Paper-Datasets_and_Benchmarks.pdf):
- Main_ontology
- Comedy_ontology
- Law_ontology
- Engineering_ontology

To download the files for these datasets, please follow the instructions found in the [official GitHub repository](https://github.com/seanyang38/Ontologue?tab=readme-ov-file#data).

We also generated 6 additional datasets following the instructions in the Ontologue paper:
- Culture_ontology
- Information_ontology
- Mathematics_ontology
- People_ontology
- Philosophy_ontology
- Energy_ontology

To download the files for these datasets, the user can use this [Google drive link](https://drive.google.com/drive/folders/1rfvNLs-v_m_xltuBKyqK3HjhGc2OyXYq?usp=drive_link).
